# Examination
> 后端面试题 
#### 要求

> 后台语言Java或python ；数据库Redis

- 对 redis 自定义 字段的访问
- 一个接口 get,post,put,delete 操作readis
- 限制ip访问次数 1 qps
- 接口发布时使用域名

#### 验证方法

1. git clone https://github.com/TX-Net/Examination.git  并进入项目目录；
2. 修改application.yml配置文件中redis的相关配置，使得能顺利连接redis环境；
3. 利用idea启动服务（默认端口为9999）即可进行测试；
